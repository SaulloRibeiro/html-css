Vamos aprender a usar textos com Semântica 😃

Usando texto em negrito

EX: Prezer meu nome é Saullo

Para fazer isso usa-se a tag STRONG

Tag: <strong></strong>

Usando texto em intálico

EX: Prazer meu nome é Saullo

Para fazer isso usa-se a tag ENPHASIS (enfâse)

Tag: <em></em>

Usando Marca texto

EX: Prazer meu nome é Saullo

EX2: Prazer meu nome é Sorocaba

EX3: Prazer meu nome é Fernandinha

Para isso usa-se a tag MARK

Tag: <mark></mark>

Textos grandes e pequenos

Os textos grandes era usado a tag 'Big' porém a W3C não considerava esssa tag Semântica, por isso ficou obsoleta
por isso se eu quero colocar um texto grande é preciso usar o css

Para usar o texto pequeno usa-se a tag SMALL

EX: Meu nome é Saullo

Tag: <small></small>

Textos deletados

Para isso usa-se a Tag DEL

EX: Ei você que está lendo, me beija?

Tag: <del></del>

Textos inseridos ou em neglito

Para isso usa-se a Tag INS

EX: Meu nome é Saullo

Tag: <ins></ins>

Texto Sobrescrito

É usado quando queremos escrever por exemplo dois ao quadro (2^2)

Para isso usa-se a Tag SUP

EX: 2 4 + 3 3 = ?

Tag: <sup></sup>

Texto Subscrito

É usado para escrever por exemplo fórmulas químicas

Para isso usa a Tag SUB

EX: H2O é a fórmula da água

Tag: <sub></sub>
